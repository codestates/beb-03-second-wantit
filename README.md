## 🌴 실행 방법

각각 필요한 모듈을 설치한다.(client / server)

npm install

env.example을 env로 수정 후 필요한 정보를 입력한다.

ganache를 실행한다.

wantit DB를 생성한다.

npx sequelize-cli db:create

npx sequelize-cli db:migrate

프로젝트를 시작한다.

npm start
